# Amazon Data-Driven Framework
## STA Mini Project 9a — Selenium + TestNG + Apache POI

### Setup (same as your previous project!)
1. Java 21 already installed ✅
2. Maven already installed ✅
3. IntelliJ already installed ✅
4. Chrome already installed ✅

### Import Steps
1. Extract AmazonDDFramework.zip to C:\Projects\
2. Open IntelliJ → File → Open → Select AmazonDDFramework folder
3. Trust Project → Wait for Maven to import
4. Right-click src/test/resources → Mark Directory As → Test Resources Root
5. File → Project Structure → SDK → temurin-21

### Run Tests
Right-click src/test/resources/testng.xml → Run 'testng.xml'
OR: mvn clean test

### Test Data
Edit testdata/AmazonTestData.xlsx to add/modify search keywords

### Dashboard
Open dashboard/index.html in browser to view test overview

### Reports
After running, open reports/AmazonTestReport_*.html in browser
